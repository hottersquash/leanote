Image Storage

Only note body images use this storage. Avatar, blog logo, albums, and attachments keep using local storage.
Do not include the bucket name. Examples: Huawei OBS obs.cn-north-4.myhuaweicloud.com, Aliyun OSS oss-cn-hangzhou.aliyuncs.com, Tencent COS cos.ap-guangzhou.myqcloud.com, AWS S3 s3.us-west-2.amazonaws.com. Both https:// and plain host are accepted.
Required for Tencent COS and AWS S3. Aliyun OSS can use region instead of endpoint (e.g. cn-hangzhou builds oss-cn-hangzhou.aliyuncs.com).
Optional. If blank, Leanote builds URLs from bucket and endpoint.
Optional. The user and date path will be appended below this prefix.
